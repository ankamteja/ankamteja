## 📁 Featured Repositories

### [🧠 network-intrusion-detection](https://github.com/ankamteja/network-intrusion-detection)
Machine learning pipeline evaluating the NSL-KDD and UNSW-NB15 datasets. Utilizes PCA and LDA dimensionality reduction alongside Random Forest, SVM, and KNN classifiers to parse anomalies in infrastructure packets.

### [🔒 CyberStegoApp](https://github.com/ankamteja/CyberStegoApp)
Android steganography application utilizing AES-256 encryption to embed secure payloads within image pixels. Features built-in session security controls, including a native shake-to-delete mechanism and internal activity logging.

### [📡 python-port-scanner](https://github.com/ankamteja/python-port-scanner)
Multithreaded TCP port scanner written in Python designed to target specified ranges on host systems and map active open infrastructure ports.
